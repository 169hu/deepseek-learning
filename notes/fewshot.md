# Few-shot（少样本学习）学习实践笔记
**日期：** 2026-07-20

## 核心理解
Few-shot 是指**在 Prompt 中提供 2-3 个完整的“输入→输出”示例**，让模型通过模仿来学习任务。它不需要训练，只需在 System Prompt 中放入示例即可。

## 我的实践
在 DeepSeek 工具箱的“信息提取”功能中，我使用了两个示例：
- 输入："小明，数学95，语文88" → 输出：{"name": "小明", "scores": {"数学": 95, "语文": 88}}
- 输入："小红，英语92，科学85" → 输出：{"name": "小红", "scores": {"英语": 92, "科学": 85}}

当用户输入从未见过的新数据时，模型能够自动模仿这两个示例的格式，输出正确的 JSON。

## 面试话术
“Few-shot 的核心是给模型看标准答案。在我的信息提取功能中，我只需要在 System Prompt 里放两个示例，模型就能准确处理任意格式的输入数据。这种方法比写正则表达式更灵活，也比微调模型更轻量。”
