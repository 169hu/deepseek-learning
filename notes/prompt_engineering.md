# Prompt Engineering 实践笔记
**日期：** 2026-07-20

## 今日实验：System Prompt 对翻译风格的影响
在 DeepSeek 工具箱的“翻译”功能中，我尝试了三种不同的系统提示词：

1. **通用翻译**：`"你是专业翻译，只输出翻译结果。"`
2. **莎士比亚风格（当前）**：`"你是莎士比亚时代的英国诗人。请将中文翻译成极具古典文学色彩的英文，不准意译，必须直译。"`
3. **带约束版本**：`"翻译成英文。如果输入包含数字，请在输出末尾加上【数字已校验】。"`

核心收获
Prompt 不仅仅是“提问”，更是“给模型设定运行规则”。同样的模型，通过改变 System Prompt，可以输出完全不同风格的答案。这说明在设计 AI 应用时，**精准的 System Prompt 是比调参更有效的控制手段。**
面试话术：
“在我的项目中，为了控制翻译的准确性，我发现‘定义角色 + 添加约束’比‘通用翻译’更精准。比如指定莎士比亚风格，会让英文输出更优雅，同时‘不准意译’的约束保证了直译的准确性。”
